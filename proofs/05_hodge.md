## Classical Formulation

### Formal Statement

For a smooth projective complex variety, every rational Hodge class is a rational combination of classes of algebraic cycles.

### Plain-Language Description

There are different ways to describe shapes in high-dimensional geometry. The conjecture says certain “nice” topological features should always correspond to actual algebraic subspaces.

### Historical Difficulty

1. **Global-to-local mapping:** cycles are global objects; classes are analytic/topological.  
2. **Representation gap:** describing a class does not produce a cycle.  
3. **Frame limitation:** local invariants can hide global constructive structure.
